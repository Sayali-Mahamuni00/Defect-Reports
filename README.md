# Defect Reports

This directory contains sample defect reports created as part of manual, functional,
and database testing practice.

## What these reports demonstrate
- Understanding of defect lifecycle
- Proper defect logging with clear title and description
- Severity and priority classification
- Expected vs Actual result validation
- Environment and test type identification
- Use of supporting screenshots

## Defect Categories
- Database Testing
- Functional Testing (Web Application)

These defect reports were created to simulate real-world QA scenarios and
showcase hands-on testing skills and documentation standards.
